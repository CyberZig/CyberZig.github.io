Network Security

My expertise in Network Security involves designing and implementing secure network architectures for both corporate IT networks and safety-critical production environments.

Key competencies:

  • Enterprise Firewall Architectures (Fortigate, Cisco)
  • IPsec and OpenVPN Site-to-Site Deployments
  • Deep Packet Inspection & Network Forensic Analysis (Wireshark)
  • Network Segregation & DMZ Configuration for CNI systems

Risk Management

Leading structured security assessments to identify, evaluate, and mitigate operational and systemic information security risks across organisational assets.

Key competencies:

  • Executing Risk Assessments aligned with ISO 27005
  • Establishing and updating Company and Product Security Risk Registers
  • Defining risk tolerance and security exception approval mechanisms
  • Performing Third-Party and Vendor Security due diligence assurance

Vulnerability Management

Running company-wide vulnerability lifecycle management, ensuring vulnerabilities are identified, prioritised, and patched to reduce the organisation's overall attack surface.

Key competencies:

  • Internal and External Vulnerability Scanning (Tenable Nessus, Qualys)
  • Coordinating Patch Management campaigns with systems engineers
  • Prioritising risk treatments based on CVSS scores and actual exploit threat intel
  • Configuring Automated Endpoint Security posture scans

Incident Response

Structuring incident response programs that outline pre-configured processes to detect, contain, and remediate actively unfolding cybersecurity threats.

Key competencies:

  • Authoring Incident Response Playbooks (Ransomware, Phishing, Data Breach)
  • Analyzing live event logs using SIEM systems
  • Orchestrating containment actions (host isolation, API key revocation)
  • Leading Post-Incident Reviews (PIR) to construct preventative controls

Digital Forensics

Analysing system memories, log file trails, and file-system artifacts to reconstruct malicious activity timelines and identify structural attack vectors.

Key competencies:

  • Performing memory-capture and filesystem preservation acquisitions
  • Auditing system event databases, registry changes, and process behaviors
  • Developing chain-of-custody protocols for digital evidence
  • Investigating indicator of compromise (IOC) pathways inside directory environments

Compliance & Certifications

Mapping administrative and technical security safeguards to critical international compliance blueprints to validate systematic governance.

Key competencies:

  • Structuring and auditing ISO/IEC 27001 Information Security Management Systems (ISMS)
  • Evaluating network environments against Cyber Essentials Plus requirements
  • Aligning CNI operations and products towards Cyber Assessment Framework (CAF)
  • Coordinating internal policy audits with external ISO registrars

Cloud Security

Configuring and validating tenant safeguards inside AWS and Microsoft Azure, reducing shared-responsibility cloud exposure risk.

Key competencies:

  • Securing cloud environments using Identity & Access Management (IAM) role boundaries
  • Managing Virtual Private Clouds (VPC), Security Groups, and virtual subnet routers
  • Evaluating tenant postures using Cloud Security Posture Management (CSPM) tools
  • Enforcing storage bucket controls and strict API endpoint verification rules

Security Training Delivery

Developing internal security cultures by running continuous interactive security programs, keeping team awareness high against modern phishing and social engineering exploits.

Key competencies:

  • Curating relevant, tailored threat briefs for technical and executive staff
  • Running scheduled, controlled phishing simulation campaigns
  • Developing developer-focused secure code baseline workshops
  • Training internal incident coordinators on initial threat reports and escalation paths
